This is the BubbleSort Algorithm.

The pseudo-code for the algorithm is as follows:
BubbleSort(A, start, end):
     swapped <- false
     while (swapped is false)
          for (i = start to end)
               if (A[i] > A[i+1]) then
                    temp <- A[i]
                    A[i] <- A[i+1]
                    A[i+1] <- temp
                    swapped <- true
               end if
          end for
     end while
end BubbleSort
